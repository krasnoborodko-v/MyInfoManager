# MyInfoManager Sidebar (React)

Фронтенд-приложение для управления информацией.

## Установка

```bash
npm install
```

## Запуск

1. Убедитесь, что бэкенд запущен на `http://localhost:8000`
2. Запустите фронтенд:

```bash
npm start
```

Приложение откроется на `http://localhost:3000`

## Настройка

Создайте файл `.env` в корне проекта:

```
REACT_APP_API_URL=http://localhost:8000
```

## Структура

```
src/
├── api/
│   └── client.js          # API клиент для связи с бэкендом
├── components/
│   ├── Sidebar.js         # Боковая панель с навигацией
│   ├── Sidebar.css
│   ├── MainPanel.js       # Основная панель
│   └── MainPanel.css
├── hooks/
│   ├── useResources.js    # Хук для работы с ресурсами
│   ├── useNotes.js        # Хук для работы с заметками
│   ├── useTasks.js        # Хук для работы с задачами
│   └── index.js
├── App.js
├── App.css
├── index.js
└── index.css
```

## Возможности

- **Ресурсы**: просмотр, создание, редактирование, удаление ресурсов и категорий
- **Заметки**: просмотр, создание, редактирование, удаление заметок и категорий
- **Задачи**: просмотр, создание, редактирование, удаление задач и категорий
- **Календарь**: просмотр календаря
- **Планировщик**: план на день
- **Настройки**: настройки приложения
